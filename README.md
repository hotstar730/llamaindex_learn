

# 运行程序
```
conda activate llama-index

flask --app llm_api_server.py
python3 llm_api_server.py
nohup python3 llm_api_server.py &
```

# 安装依赖包
```
pip install -r requirements.txt
```

# 安装mysql包需要安装
```
yum install python3-devel mysql-devel pkgconfig
```
